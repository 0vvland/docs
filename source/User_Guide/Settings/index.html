--- layout: page weight: 100 title: Settings seo: title: Settings navigation: show: true --- {% info %} This page refers to our beta UI functionality. To find these features, please click the “Beta” button at the top of the SendGrid Customer Portal. {% endinfo %} SendGrid allows you to configure and control everything about your SendGrid account. All of the settings that you can manage through the User Interface can also be managed through the API v3. {% anchor h2 %} Account Details {% endanchor %}

Your basic account settings for username, password, email address, and business address details.

{% anchor h2 %} Alert Settings {% endanchor %}

Receive alerts about your account and your SendGrid behavior via email.

{% anchor h2 %} API Keys {% endanchor %}

Create and manage your API Keys, which allow you to use SendGrid from your application or website.

{% anchor h2 %} Billing Settings {% endanchor %}

Manage your account and payment settings

{% anchor h2 %} Inbound Parse {% endanchor %}

Send and push emails to SendGrid, which will be broken apart by our software and returned to you as data that you can use for your application.

{% anchor h2 %} Mail Settings {% endanchor %}

Control how SendGrid filters and processes your emails as they go out to your recipients.

{% anchor h2 %} Partners {% endanchor %}

Control settings for SendGrid partner services.

{% anchor h2 %} Subuser Management {% endanchor %}

Create and manage subusers to segment your email sending and have different settings for your segments.

{% anchor h2 %} Tracking {% endanchor %}

Control how your emails and your recipient email interaction behaviors are tracked within SendGrid.
